Principais Formatações

Negrito/Destaque

Nessa frase temos um Termo em Negrito/Destaque usando a tag "< b >" (não semântica) (n)

Nesta frase temos um Termo em Negrito/Destaque usando a tag "< strong >"(semântica)>

Itálico/Ênfase

Nesta frase temos um Termo em Itálico usando a tag "< i >"

Nesta frase temos um Termo em Itálico usando a tag EM -> Ctrl+Shift+P - (semântica)

Podemos criar um texto marcado (destacado por marca texto), usando a TAG "< mark >".

Vamos destacar o texto com a cor violeta.

Texto Grande e pequeno

Estamos criando um texto grande e um texto pequeno na mesma frase, com as tags < big >(obsoleto) e < small > (nao obsoleto)

Texto Deletado

Podemos marcar um texto como excluido com a tag < del > para indicar que ele deve ser lido, mas nao considerado

Texto Inserido

Podemos marcar um texto como inserido com a tag < ins > para dar ênfase e indicar que ele foi adicionado depois

Existe tambem o texto sublinhado com a tag < u > (nao semantica).

Texto Sobrescrito

Para inserir letras e/ou numeros que ficam acima de algum texto/numero, exemplo: x2 (ao quadrado) + y3 (ao cubo) ficaria = x2 + y3

Texto Subscrito

Para inserir letras e/ou numeros que ficam abaixo de algum texto/numero, exemplo: H2O ficaria = H2O
